■フォントご使用上の注意とお願い
すべてのフォントの著作権は、DigitalDreamDesignに帰属します。無断転載、再配布は固く禁止させていただきます。
雑誌等に掲載する場合は必ず事前にご連絡くださいますようお願いいたします。

◆フリーフォントについて
個人使用（非営利目的のHPでのロゴ、ボタン、フリー素材など）はフリーで使っていただいて結構ですが、商用使用（法人使用の一切）の場合は、必ず事前に連絡くださいますようお願いいたします。
◆シェアフォントについて
個人使用、商用使用ともに有料となります。

これらのフォントをインストールして万が一支障が発生した場合でも、一切の責任を負いかねますのでご了承ください。

■カナ入力について
カタカナ、ひらがなフォントの入力における"KEY MAP"は当サイトの"FONT"ページの"KEY MAP"をご参照ください。
